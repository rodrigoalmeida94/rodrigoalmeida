### Project Overview
Developed an ingestion pipeline and search API capable of handling millions of satellite images. The system is designed to return similarity results at scale, enabling researchers to find related imagery efficiently.

### Technical Details
- **Data Ingestion**: High-throughput pipeline for processing millions of NASA Earthdata images.
- **Search API**: Scalable API for similarity search across massive datasets.
- **Scalability**: Designed for high-performance retrieval and processing.
- **Data Source**: Integrated with NASA Earthdata archive.
